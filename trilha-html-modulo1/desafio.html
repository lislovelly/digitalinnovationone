O que são tags HTML?

(voltar)

As tags HTML são elementos fundamentais para criar páginas na web. Elas marcam e definem a estrutura e o conteúdo visual de uma página. Vamos explorar algumas informações importantes sobre tags HTML:

  • As tags são usadas para informar ao navegador a estrutura do site.
  • Elas são sempre delimitadas pelos sinais de chevron (< e >).
  • Existem dois tipos de tags: as que precisam de fechamento (sintaxe <.tag> ) e as que não precisam (sintaxe <.tag/>).
  • As tags podem ter atributos que modificam sua estrutura ou funcionalidade.

Quais são as principais tags HTML?

(voltar)

Veja abaixo as tags mais comuns:

  • <.html>":Tag que limita onde o código começa e termina."
  • <.head>":Tag que define o começo e o fim do cabeçalho de um documento HTML."
  • <.meta>":Tag traz dados básicos sobre o documento, como autor e data de criação."
  • <.body>":Tag que indica para o navegador o corpo do documento HTML."
  • <.section>":Tag utilizada para definir seções em um documento."
  • <.article>":Tag que define um conteúdo independente e completo de outros"
  • <.div>":Tag que define um bloco inteiro de conteúdo."
  • <.title>":Tag que define o título do documento HTML. Esse é o nome que é exibido no topo da janela do navegador quando a página é aberta."
  • <.h1>..<.h6>":Definem o tamanho do título do documento, sendo <.h1> de maior valor hierárquico e <.6> de menor valor hierárquico."
  • <.p>":Tag que define o começo e o fim de um parágrafo."
  • <.i>,<.u>":Tag <.i> formata textos em itálico e a tag <.u> em sublinhado."
  • <.br>":Tag utilizada para quebra de linha em um texto."
  • <.strong>":Tag utilizada para que uma parte do texto fique em negrito."
  • <.font>":Tag que alterna tamanho, cor e tipo de fonte exibida."
  • <.span>":Tag utilizada para agrupar elementos para fins de estilo."
  • <.a>":Tag que cira um link e atribui elementos a ele como: href, name, target, rel, entre outros."
  • <.ol>":Tag que cria listas ordenadas."
  • <.ul>":Tag que cria listas não ordenadas."
  • <.li>":Tag Define cada item da lista, que fazem parte de listas ordenadas ou não ordenadas."
  • <.sub>":Tag que exibe texto como subscrito."
  • <.sup>":Tag que exibe texto como sobrescrito."
  • <.blockquote>":Tag que indica que o texto incluído é uma citação longa."
  • <.abbr>":Tag que defiene uma abreviação ou acrônimo no texto."
  • <.del>":Tag que representa uma parte do texto que foi excluída de um documento."
  • <.mark>":Tag é usada para destacar partes do texto."
  • <.abbr>":Tag representa uma abreviação e opcionalmente fornece uma descrição completa para ela."
  • <.small>":Tag é utilizada para definir texto menor."
  • <.del>":Tag é usado para representar conteúdo que foi excluído ou riscado."

Conclusão

(voltar)

Arquivos HTML necessitam de tags para organizar o documento, pois quando o navegador da Web vê uma tag ele sabe exatamente o que fazer para exibir o conteúdo na tela. As tags diferenciam um conteúdo HTML de um texto simples pois elas possuem elementos para estruturar o arquivo e atributos que fornecem informações necessárias a esse elemento, sendo muitos deles de uso obrigatório para que possa ser definido as características dos elementos.

Documentos HTML e suas características são criados e renderizados usando tags HTML, e por padrão, as tags HTML são palavras ou letras específicas, escritas dentro de colchetes angulares ( <> ).

Mesmo com o uso de tags básicas, precisamos entender que elas não dão nenhuma indicação sobre o tipo de conteúdo que contêm ou qual papel que o conteúdo desempenha na página, e para isso fazemos uso da semântica no HTML onde abordamos essa deficiência definindo tags específicas para indicar claramente qual papel desempenhado pelo conteúdo que essas tags contêm.
